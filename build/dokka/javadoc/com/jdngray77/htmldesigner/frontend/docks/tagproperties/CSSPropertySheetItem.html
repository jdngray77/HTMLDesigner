Class CSSPropertySheetItem

  • All Implemented Interfaces:
    org.controlsfx.control.PropertySheet.Item 
    
public class CSSPropertySheetItem
 implements PropertySheet.Item

    Abstract representation of a css property.

    This is an item that can be added to a PropertySheet to represent an editable CSS property for a given html element.

    When it receives the input from a CSSPropertyEditor, it handles updating the value in the Element's style attribute.

    i.e

    Edited with a CSSPropertyEditor of some kind.

    Can be used alone for simple properties that, but can be extended for CSS entries that require custom implementations or requirements.
