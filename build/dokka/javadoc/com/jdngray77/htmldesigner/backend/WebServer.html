public class WebServer
 implements Subscriber, Restartable

    A (Live-ish) web server for development and debugging.

    Hosts the current document, so it can be viewed in a real-world web browser.

    Generally the only user-feedback of this system is some notifications show at start and stop.

         final Boolean getAutoRefresh()
